# CALM — Cognitive Apprenticeship via LLMs

Most AI tools give students direct answers. CALM does not.

CALM is a calculus tutoring system that forces students to think. Instead of solving problems for them, it asks questions, breaks concepts down step by step, and only moves forward when the student demonstrates understanding. Every explanation is pulled directly from Thomas's Calculus (14th edition), so nothing is fabricated.

The system tracks each student's mastery score in real time using Bayesian Knowledge Tracing. As the score rises, the difficulty increases across seven levels — from basic analogies to graduate-level problems. If a student answers correctly too quickly, the system assumes a guess and adjusts. If they struggle, it steps back and rebuilds from the foundation.

The goal is one-on-one expert tutoring at near-zero cost.
